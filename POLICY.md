# Privacy Policy — GDDL Demon Recommender
Last updated: April 2026

GDDL Demon Recommender is a Chrome extension that recommends 
Geometry Dash demons from gdladder.com based on your skill profile.

## DATA COLLECTED AND HOW IT IS USED

When you are logged in to gdladder.com, the extension reads the 
following data from the GDDL public API using your existing browser 
session:

  • Your GDDL user ID and submission count — used only to detect 
    when your beaten-level list has changed and the local cache 
    needs to be refreshed. This is stored on your device in 
    chrome.storage.local and is never transmitted off your device.

  • Your beaten level IDs and aggregated skill scores — transmitted 
    to the recommendation server at gddl.melvinwhitaker.com as part 
    of a recommendation query. This data is used solely to compute 
    and return results for that request. It is not logged, stored, 
    or associated with any user account on the server.

  • The ID of the level page you are currently viewing — transmitted 
    to the recommendation server to find similar levels. Same 
    handling as above.

When you are not logged in, only the current level ID is used.

## DATA SHARING

No data is shared with third parties. The only external server this 
extension communicates with is gddl.melvinwhitaker.com (the 
recommendation backend operated by the developer) and the public 
GDDL API at gdladder.com.

## CHILDREN'S PRIVACY

This extension does not knowingly collect personal information from 
children under the age of 13.
